《**乌云典当记**》是[万能青年旅店](https://zh.wikipedia.org/wiki/万能青年旅店)的第二张EP专辑，只包含一首歌。

## 介绍

该歌作为智利政治电影《[NO](https://zh.wikipedia.org/wiki/向政府說不)》在[台湾](https://zh.wikipedia.org/wiki/台湾)上映的中文主题曲，这张单曲唱片仅在台湾地区发行，未来亦无再版计划。
