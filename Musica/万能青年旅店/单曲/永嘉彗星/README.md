2023年初，在第二张专辑《冀西南林路行》发布后的第三个年头，万能青年旅店将此前已现端倪的一首乐曲命名为《永嘉彗星》，出现在了他们的演出中。

“永嘉彗星”中，“永嘉”是司马炽在位时的年号。晋怀帝司马炽即位次年，改元永嘉，这一年号一直用到司马炽遭毒杀身亡，一共六年。

中国文化中，年份与星座并置，其含义便由天象指向了人事。
